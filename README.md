# 📱 Google Play Store Analytics Dashboard

An end-to-end Data Analytics project built using **Python, SQL, Excel, and Power BI** to analyze Google Play Store applications and user reviews. The project demonstrates the complete analytics workflow—from raw data preprocessing to interactive business dashboards.

---

## 📌 Project Overview

This project analyzes over **10,000 Google Play Store applications** to discover trends in app categories, installs, ratings, pricing models, audience segmentation, Android versions, and user reviews.

The objective is to transform raw application data into meaningful business insights using a complete analytics pipeline.

## 📌 Key Business Insights

- Family and Game categories contain the highest number of applications.
- More than **90%** of Play Store applications are Free.
- The "Everyone" audience category dominates the Play Store.
- Android 4.1+ has the largest number of supported applications.
- App installations increased significantly between 2016 and 2018.
- Educational and Game applications maintain higher average ratings.
- User sentiment analysis highlights apps with positive and negative customer feedback.
